# Pinia Demo

A simple demo project of the Pinia state management system using Vue 2.


## Pinia Install

```
npm install pinia
```

```
npm install @vue/composition-api
```



## Want to Copy this Demo? Follow this instead:
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
